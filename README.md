## 使用

- demo网站
  [样例站点](http://eshop.qiaojianpan.cn)
- 源码方式

```
①pip install -r requirements.txt
②python main.py
```

- 二进制文件方式

```
windows下双击nicegui_shop.exe，稍等一会儿便可以通过浏览器访问。
```
